- 从指定 URL 获取内容并处理
- 接受 URL 和可选格式作为输入
- 获取 URL 内容，转换为请求的格式（默认 Markdown）
- 当你需要检索和分析网页内容时使用此工具

用法说明：
  - 重要：如果有其他工具提供更好的网页获取能力、更针对任务或限制更少，优先使用那个工具
  - URL 必须是格式完整的有效 URL
  - HTTP URL 会自动升级为 HTTPS
  - 格式选项："markdown"（默认）、"text" 或 "html"
  - 此工具是只读的，不修改任何文件
  - 如果内容非常大，结果可能被摘要
  - 当 URL 重定向到不同主机时，工具会通知你并提供重定向 URL。你应使用重定向 URL 发起新的请求
  - 对于 GitHub URL，优先使用 Bash 中的 gh CLI（例如 gh pr view、gh issue view、gh api）
